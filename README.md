# Retrograde Input 16

This requires the use of [Retrograde Image](https://github.com/retrograde-dev/retrograde-image) to generate usable output.

Pre-generated output is available as a free download at [https://retrograde-dev.itch.io/retrograde-input-16](https://retrograde-dev.itch.io/retrograde-input-16).

## Godot Addon

You can also use the [Godot Addon](https://github.com/retrograde-dev/retrograde-image-godot-addon) to generate images directly in your project.
